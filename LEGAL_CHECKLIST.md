# Rechtliche Go-Live-Checkliste Deutschland/EU

Stand: Demo erstellt am 02.06.2026

## 1. Unternehmensdaten

- [ ] Gewerbe angemeldet
- [ ] Steuernummer / steuerliche Erfassung geklärt
- [ ] Kleinunternehmerregelung oder Umsatzsteuerpflicht entschieden
- [ ] Anbietername korrekt
- [ ] ladungsfähige Anschrift eingetragen
- [ ] geschäftliche E-Mail-Adresse eingetragen
- [ ] Telefonnummer geprüft/entschieden

## 2. Impressum

- [ ] Impressum leicht erkennbar, unmittelbar erreichbar und ständig verfügbar
- [ ] vollständiger Name / Firma
- [ ] Anschrift
- [ ] E-Mail
- [ ] USt-ID nur angeben, wenn vorhanden
- [ ] keine private Daten versehentlich falsch oder unvollständig

## 3. Produkt und Preis

- [ ] Produktbeschreibung klar und nicht irreführend
- [ ] kein Versprechen wie „rechtssicher“, „gerichtsfest“, „garantiert bezahlt“
- [ ] Gesamtpreis sichtbar
- [ ] Hinweis auf digitalen Download
- [ ] Versandkosten 0,00 € oder „kein Versand“
- [ ] Steuerhinweis korrekt: Kleinunternehmer oder Umsatzsteuer

## 4. Checkout

- [ ] Button heißt „zahlungspflichtig bestellen“ oder eindeutig gleichwertig
- [ ] AGB verlinkt
- [ ] Datenschutzerklärung verlinkt
- [ ] Widerrufsbelehrung verlinkt
- [ ] digitale Inhalte: ausdrückliche Zustimmung zur sofortigen Bereitstellung
- [ ] digitale Inhalte: Bestätigung des Verlusts des Widerrufsrechts
- [ ] Checkboxen nicht vorausgewählt
- [ ] Bestellbestätigung per E-Mail
- [ ] Zustimmung zum Widerrufsverlust wird protokolliert

## 5. Widerruf

- [ ] Widerrufsbelehrung aktuell
- [ ] Muster-Widerrufsformular bereitgestellt
- [ ] digitale Inhalte korrekt behandelt
- [ ] ab 19.06.2026: Widerrufsfunktion/„Vertrag widerrufen“ sichtbar und leicht zugänglich
- [ ] Widerrufsfunktion zweistufig
- [ ] elektronische Eingangsbestätigung mit Datum/Uhrzeit
- [ ] nur notwendige Daten abfragen

## 6. Datenschutz

- [ ] Datenschutzerklärung auf tatsächliche Tools angepasst
- [ ] Hosting-Anbieter eingetragen
- [ ] Zahlungsanbieter eingetragen
- [ ] Kontaktformular-Dienst eingetragen
- [ ] keine Tracking-Tools ohne Consent
- [ ] Consent-Banner, falls Marketing/Analyse-Cookies eingesetzt werden
- [ ] Auftragsverarbeitungsverträge prüfen

## 7. Streitbeilegung

- [ ] Hinweis zur Verbraucherstreitbeilegung geprüft
- [ ] alten EU-OS-Link entfernen, da Plattform seit 20.07.2025 eingestellt ist

## 8. Technisches

- [ ] Domain mit SSL/HTTPS
- [ ] geschäftliche E-Mail mit Domain
- [ ] Backups
- [ ] Downloadlink geschützt
- [ ] Rechnungen und Buchhaltung organisiert
- [ ] Testkauf durchgeführt

## 9. Arbeitgeber / Nebengewerbe

- [ ] Arbeitsvertrag auf Nebentätigkeitsklausel geprüft
- [ ] Nebentätigkeit ggf. angezeigt/genehmigt
- [ ] keine Firmenfotos, Kundendaten, Baustellendaten oder Arbeitgeber-Vorlagen verwenden
- [ ] Website-Arbeit nur außerhalb der Arbeitszeit
