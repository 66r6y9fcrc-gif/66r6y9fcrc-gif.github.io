Einrichtung (einmalig) & Datenschutz

1. Kostenloses Projekt auf supabase.com anlegen.
2. Links im SQL Editor diesen Code ausführen:

create table if not exists brains (
  id text primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table brains enable row level security;
create policy "zugriff per schluessel"
  on brains for all using (true) with check (true);

3. Unter Project Settings → API die Project URL und den anon public Key kopieren und oben einfügen.
4. Einen geheimen Gehirn-Schlüssel wählen und auf jedem Gerät identisch eintragen.

Datenschutz: Dein Gehirn ist durch den geheimen Schlüssel geschützt. Wer URL, Key und Schlüssel kennt, kann die Daten lesen. Für rein persönliche Nutzung mit einem schwer zu erratenden Schlüssel ist das in Ordnung; ein echter Login pro Person wäre ein späterer Ausbau.
